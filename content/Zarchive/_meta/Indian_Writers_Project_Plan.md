**4. Standardized Structure for Each Writer:**

For each writer, the following directory and file structure will be created inside `Literature/Indian Writers/`:

```
[Writer Name]/
├── 1_Biography.md
├── 2_Works_and_Awards.md
├── 3_Major_Works_Analysis/
│   ├── [Work 1 Name].md
│   ├── [Work 2 Name].md
│   ├── [Work 3 Name].md
│   ├── [Work 4 Name].md
│   └── [Work 5+ Name].md
├── 4_Literary_Style_and_Themes.md
├── 5_Mindmap.md
└── 6_Practice_MCQs.md
```

**5. Content Requirements for Each File:**

*   **`1_Biography.md`**: A detailed narrative of the writer's life, synthesizing all sources. Will include early life, education, career, personal life, influences, mentors, and contemporaries.
*   **`2_Works_and_Awards.md`**: A complete catalog of the writer's works, organized by genre. Each entry will detail the title, publication year, a brief synopsis, and any awards received.
*   **`3_Major_Works_Analysis/`**: A dedicated folder containing separate, in-depth analyses for at least five major works. Each file will cover:
    *   Detailed Plot Summary
    *   In-depth Character Analysis
    *   Exploration of Major Themes and Symbols
    *   Critical Reception and Legacy
    *   A list of important quotes.
*   **`4_Literary_Style_and_Themes.md`**: A dedicated file analyzing the writer's overall literary contribution, including:
    *   Dominant themes across their works.
    *   Signature writing style and use of literary devices.
    *   Their unique contribution to Indian English Literature.
*   **`5_Mindmap.md`**: A markdown file formatted for `markmap` to create a visual, hierarchical overview of the writer's life, major works, themes, and key relationships.
*   **`6_Practice_MCQs.md`**: A file containing at least 100 original multiple-choice questions based on all compiled information. Answers will be hidden within a `<details>` toggle for self-assessment.
